# LinuxShell_route


* 查詢路由設定

usage

      route [args] cmd [[modifiers] args]
      
 * args
 
    （略）
 

* cmd

    * add: 增加路由紀錄。
    
    * del: 刪除路由紀錄。
    
    * -net: 指定網段位址。
    
    * -host: 指定主機位址。
    
    * target: 目的地位址：
      

* the arg for cmd

          +--------+---------------------
          | dev IF | 指定路由僅跟特定裝置
          +--------+
